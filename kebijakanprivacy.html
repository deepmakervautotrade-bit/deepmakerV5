Kebijakan Privasi

Edisi terakhir: 16 Februari 2026

1. Pendahuluan

Deepmaker V5.5 ("kami", "tim", "layanan") berkomitmen untuk melindungi privasi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, mengungkapkan, dan melindungi informasi pribadi Anda saat Anda mengunjungi situs web kami, mendaftar, membeli lisensi, menggunakan Expert Advisor (EA), atau berinteraksi dengan layanan kami. Dengan menggunakan layanan kami, Anda menyetujui pengumpulan dan penggunaan informasi sesuai dengan kebijakan ini.

2. Data yang Kami Kumpulkan

Kami dapat mengumpulkan beberapa atau semua informasi berikut:

a. Informasi Akun: Nama lengkap, alamat email, nomor telepon (opsional), kode lisensi, dan detail pembayaran (diproses oleh Midtrans, kami tidak menyimpan data kartu kredit).

b. Informasi Teknis EA: Saat EA berjalan, kami dapat menerima data berikut untuk keperluan validasi, pemantauan, dan peningkatan layanan: ID akun MT5, nama broker, server, leverage, balance, equity, mata uang akun, status EA (ON/OFF), dan device ID (nomor akun trading).

c. Data Penggunaan Situs: Alamat IP, jenis browser, halaman yang dikunjungi, waktu akses, dan data analitik lainnya melalui cookie atau teknologi serupa.

d. Komunikasi: Setiap korespondensi yang Anda kirimkan kepada kami melalui email, chat, atau formulir kontak.

3. Tujuan Penggunaan Data

Kami menggunakan data yang dikumpulkan untuk:

  • Memproses pendaftaran, pembayaran, dan pengiriman lisensi.
  • Memvalidasi lisensi EA dan mencegah penggunaan tidak sah (device binding).
  • Memantau kinerja EA, memperbaiki bug, dan mengembangkan fitur baru.
  • Menyediakan dashboard multi-user yang menampilkan informasi akun dan transaksi Anda.
  • Mengirimkan pemberitahuan penting terkait pembaruan, pemeliharaan, atau perubahan kebijakan.
  • Mematuhi kewajiban hukum dan mencegah penipuan.

Catatan khusus: Data akun trading (balance, equity) digunakan semata-mata untuk ditampilkan di dashboard Anda dan untuk keperluan analisis internal. Kami tidak pernah menjual atau membagikan data ini kepada pihak ketiga tanpa persetujuan Anda, kecuali diwajibkan oleh hukum.

4. Keamanan Data

Kami menerapkan langkah-langkah keamanan teknis dan organisasi yang sesuai untuk melindungi data pribadi Anda dari akses tidak sah, perubahan, pengungkapan, atau perusakan. Semua data disimpan di database dengan akses terbatas hanya untuk tim pengembang. Komunikasi antara EA dan server dienkripsi menggunakan HTTPS. Namun, tidak ada metode transmisi melalui internet atau penyimpanan elektronik yang 100% aman, sehingga kami tidak dapat menjamin keamanan absolut.

5. Penggunaan VPS dan Data Real-time

Untuk menjalankan EA secara optimal, Anda diwajibkan menggunakan VPS. Dalam mode tersebut, EA akan secara periodik mengirimkan data akun ke server kami. Data ini hanya digunakan untuk keperluan tampilan dashboard dan tidak akan dibagikan kepada pihak lain. Jika Anda memilih untuk tidak menggunakan VPS, EA mungkin tidak dapat berfungsi dengan baik, dan kami tidak bertanggung jawab atas ketidakakuratan data akibat koneksi yang tidak stabil.

6. Cookie dan Teknologi Serupa

Situs web kami dapat menggunakan cookie untuk meningkatkan pengalaman pengguna, menyimpan preferensi, dan menganalisis lalu lintas. Anda dapat mengatur browser untuk menolak semua cookie atau memberi peringatan saat cookie dikirim. Namun, beberapa bagian situs mungkin tidak berfungsi dengan baik jika cookie dinonaktifkan.

7. Hak Anda

Anda memiliki hak untuk:

  • Mengakses data pribadi yang kami simpan.
  • Meminta koreksi data yang tidak akurat.
  • Meminta penghapusan data, kecuali data yang wajib kami simpan untuk kepentingan hukum atau keamanan.
  • Menarik persetujuan penggunaan data kapan saja.

Untuk melaksanakan hak Anda, silakan hubungi kami melalui email atau live chat yang tersedia di situs.

9. Perubahan Kebijakan Privasi

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Perubahan akan diumumkan melalui situs web dan, jika signifikan, melalui email. Tanggal revisi terbaru akan selalu ditampilkan di bagian atas halaman ini. Dengan terus menggunakan layanan setelah perubahan, Anda dianggap menyetujui kebijakan yang diperbarui.
